# Titanic Data Analysis

## Project Overview

This project analyzes the Titanic dataset using Python. The dataset was cleaned, explored, and visualized to understand the factors affecting passenger survival.

## Objectives

- Load the Titanic dataset
- Clean missing values
- Analyze passenger survival
- Compare survival based on gender
- Compare survival based on passenger class
- Analyze survival by age group
- Visualize the data using Matplotlib and Seaborn

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Dataset

The dataset used in this project is the Titanic dataset from Kaggle.

## Project Files

- `Titanic_Data_Analysis.ipynb` – Complete Jupyter Notebook
- `README.md` – Project documentation

## Key Findings

- Female passengers had a higher survival rate than male passengers.
- First-class passengers had the highest survival chances.
- Passenger age influenced survival patterns.
